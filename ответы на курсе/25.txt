public class Person {
    private String name;
    private int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    @Override
    public boolean equals(Object obj) {
        if (this == obj) return true;                 // 1. Сравнение с самим собой
        if (obj == null || getClass() != obj.getClass()) return false; // 2. Проверка типа
        Person person = (Person) obj;                 // 3. Приведение
        return age == person.age && Objects.equals(name, person.name); // 4. Сравнение полей
    }
}

Соглашения, которым должна следовать реализация equals() (из спецификации Java):
Рефлексивность – объект должен быть равен самому себе: x.equals(x) возвращает true.
Симметричность – если x.equals(y) истинно, то и y.equals(x) должно быть истинно.
Транзитивность – если x.equals(y) истинно и y.equals(z) истинно, то x.equals(z) тоже истинно.
Постоянство – многократный вызов equals() для одних и тех же объектов должен возвращать одинаковый результат (при неизменности полей).
Сравнение с null – x.equals(null) всегда возвращает false.
